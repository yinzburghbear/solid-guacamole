# Graphite Theme for Stash

A lightweight dark CSS theme for Stash 0.31.x.

## Install

1. Copy `graphite.yml` and the `graphite` folder into:

   `%USERPROFILE%\.stash\plugins`

2. In Stash, go to **Settings → Plugins**.
3. Click **Reload plugins**.
4. Enable/use **Graphite Theme**.
5. Hard refresh the browser with `Shift + F5` if needed.

## What it changes

- Dark graphite color system
- Figtree / modern system font stack
- Softer cards, dialogs, buttons, and inputs
- Reduced wasted spacing
- Cleaner hover states
- Better tag pills and metadata readability
- No heavy JS, no backdrop-filter nonsense, no GPU tantrums

## Notes

This is CSS-only. Some Stash components may need selector tweaks depending on future Stash updates.
