# Graphite Theme 1.0.0-alpha9-src

This package is Graphite alpha9 split into source partials.

- Runtime CSS: `graphite/graphite.css`
- Source entrypoint: `graphite/src/index.css`
- Build script: `python tools/build-css.py`

Stash still loads the compiled `graphite/graphite.css`, so this version should behave the same as alpha9 while being easier to edit.
